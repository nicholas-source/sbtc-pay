sBTC Pay Widget SDK — demo

This page demonstrates the script-tag integration. Drop one <script> plus a data-sbtcpay element and you have a working Pay button.

Declarative — direct payment

<div data-sbtcpay="direct"
     data-sbtcpay-merchant="SP1ABC...XYZ"
     data-sbtcpay-amount="100000"
     data-sbtcpay-memo="Coffee"></div>

Renders below — click to open the payment modal. (Demo uses a placeholder merchant; clicking Pay inside the modal will return "merchant not found" until you swap in a real registered principal.)

Declarative — invoice payment

<div data-sbtcpay="invoice" data-sbtcpay-invoice="1"></div>

Custom label + programmatic open

<div data-sbtcpay="direct" data-sbtcpay-merchant="..."
     data-sbtcpay-amount="500000" data-sbtcpay-label="Buy me a coffee ☕"></div>

<button id="js-open">Custom checkout button</button>
<script>
  document.getElementById('js-open').addEventListener('click', () => {
    SBTCPay.open({
      mode: 'direct',
      merchant: 'SP3SD2DG6E03G3B7H6C2QAB78QKWN5WS28QKWSY1Z',
      amount: 200000,
      memo: 'From custom button',
      onSuccess: ({ txId }) => logEvent('onSuccess txId=' + txId),
      onError:   ({ message }) => logEvent('onError ' + message),
      onClose:   () => logEvent('onClose'),
    });
  });
</script>
